# ✋ Detector de Mão em Python
Projeto simples de detecção de mãos em tempo real, desenvolvido como parte dos meus estudos sobre **automação** com Python e visão computacional.
 
## 📋 Sobre o projeto
Este é um script básico que utiliza a webcam para detectar mãos em tempo real, identificando pontos de referência (landmarks) e possibilitando o rastreamento de gestos. O objetivo principal foi entender os fundamentos de visão computacional aplicados à automação de tarefas.
 
## 🚀 Tecnologias utilizadas
- [Python 3](https://www.python.org/)
- [OpenCV](https://opencv.org/) — captura e processamento de imagem
- [MediaPipe](https://developers.google.com/mediapipe) — detecção dos landmarks da mão

## 🎯 Funcionalidades
- Captura de vídeo em tempo real pela webcam
- Detecção e marcação dos pontos da mão
- Exibição do FPS (quadros por segundo)

## 📚 O que aprendi com esse projeto
- Fundamentos de processamento de imagem com OpenCV
- Uso de modelos pré-treinados de detecção com MediaPipe
- Manipulação de vídeo em tempo real
- Primeiros passos com automação baseada em visão computacional
